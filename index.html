🔥 如何接入 Firebase（5 步驟）

① 建立 Firebase 專案
前往 console.firebase.google.com，新增專案
② 啟用 Realtime Database
選擇「以測試模式開始」，取得資料庫 URL
③ 替換連線設定
將頁面 JS 中的 FIREBASE_URL 換成你的資料庫 URL
④ 寫入票數用 Transaction（防止競爭）
db.ref('votes/候選人').transaction(v => v + 1)
⑤ 設定 Security Rules
讓 votes/ 公開可讀，vote_records/ 僅管理員可讀
